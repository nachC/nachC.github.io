Dublin Bus Alarm

  • Android
  • Java
  • Firebase Realtime Database
  • GoogleMaps API
  • Python

My dearest project (ongoing). When I used to live in Dublin, I would take the bus to and from work, and quite often I'd see people sleeping on the bus. I wondered how did the passengers know when to wake up to get out? Let me tell you, Dublin Bus service is not known for it's punctuallity. Maybe, there was a problem to fix! We usually use time based alarms to wake up, but here, time is not an invariant. So, what doesn't change regarding a bus route? The route itself! The sequence of stops will be the same, regardless of how long it takes for the bus to do it. Let's set an alarm based on location! And that's what Dublin Bus Alarm (I'm great with names, I know) does.
I would say my biggest challenge with this project was when the RESTful API I was using was deprecated and replaced with GTFS. As I only needed static data (no realtime data), I decided to parse all the GTFS files and extract everything that I needed regarding the routes, and structure that data in a way that retrieving it form Firebase would be easy.
Find out more about this project on it's GitHub repo!

ImportEmail

  • Python
  • Gmail API
  • PySimpleGUI

This is an attempt to be more lazy and efficient at work. This script allows me to import/insert an email message into my Gmail account by just authenticating and selecting the file. It will do the encoding to base64_url using base64 lib, and creating the API request and calling the Gmail API, which are the steps that take the most time.
To make it even faster I also added a GUi (PySimpleGUI) that allow me to browse the .eml or .txt file containing the email and import it with one click.
